Qu'est ce que Movie Card ?


Movie Card est un projet étudiant réalisé par Julien Da Costa et Eliott Demont dans le cadre d'une semaine dédiée aux API en deuxième année de Bachelor Web à l'IIM. Sur ce site, vous pouvez visualiser quels sont les films les plus populaires, mais vous pouvez aussi visualiser les notes attribuées à un film spécifique de votre choix.


Pour la réalisation de ce site, les API TMDb (films populaires) et OMDb ont (notes des films) ont été utilisées


Bonne visite sur notre site !
